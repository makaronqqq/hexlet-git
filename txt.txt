git status показывает что сейчас идет в коммит, какие изменения сделаны
cd переходит в папку, cd .. переходит на папку вверх
git add . добавляет все изменения, можно добавить папку или файл или все файл в папке и тд
git branch показывает на какой мы ветке и какие ветки есть
git commit -m 'коментарий'  коммитит добавленный верез add изменения
git push отпрваляет на гитхаб версию с последнего коммита
git restore восстанавливает удаленный файл из репозитория (папки .git)
rm удаляет файл
git rm удаляет файл и подготавливает его к коммиту (add не надо для него прописывать)
git diff выводит все изменения в файлах --staged показывает добавленные файлы, чот не совсем разобрался в этой команде...
mv переименование файла
git log показывает все коммиты, но почему то не все :) --online чтобы показывался в 1 строку и больше коммитов видно. флан -p выводит дифф для каждого коммита
git blame <путь до файла>Эта команда выводит файл и рядом с каждой строкой показывает того, кто её менял и в каком коммите.
