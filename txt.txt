git status показывает что сейчас идет в коммит, какие изменения сделаны
cd переходит в папку, cd .. переходит на папку вверх
git add . добавляет все изменения, можно добавить папку или файл или все файл в папке и тд
git branch показывает на какой мы ветке и какие ветки есть
git commit -m 'коментарий'  коммитит добавленный верез add изменения
git push отпрваляет на гитхаб версию с последнего коммита
git restore восстанавливает удаленный файл из репозитория (папки .git)
rm удаляет файл
git rm удаляет файл и подготавливает его к коммиту (add не надо для него прописывать)
git diff выводит все изменения в файлах --staged показывает добавленные файлы, чот не совсем разобрался в этой команде...
mv переименование файла
git log показывает все коммиты, но почему то не все :) --online чтобы показывался в 1 строку и больше коммитов видно. флан -p выводит дифф для каждого коммита
git blame <путь до файла>Эта команда выводит файл и рядом с каждой строкой показывает того, кто её менял и в каком коммите.
gip grep <text> ищет воспадения с поискком, но чот не совсем у меня работает 
git clean -fd удаляет не закомиченный файлы (вероятно еще изменения, но чот хз) из текущей директории, отменить никак нельзя, аккуратней.
git restore <Имя файла/вероятно можно и всю папку> отменяет незакоммиченные изменения, возвращает файл в исходное состояние, отменить нельзя
git restore --staged <Имя файла> убирает файл из подготовки к коммиту
git revert <коммит> отменяет изменения сделанные в коммите, создает новый, обратный, коммит
Git reset удаляет коммит. ВАЖНО! УДАЛЯТЬ СТОИТ ТОЛЬКО ПОСЛЕДНИЙ КОММИТ, КОТОРЫЙ ЕЩЕ НЕ БЫЛ ЗАЛИТ НА ГИТХАБ, ИНАЧЕ СЛОМАЕТСЯ СИСТЕМА У ОСТАЛЬНЫХ ПРОГЕРОВ. git reset --hard head~ Флаг --hard означает полное удаление. Без него git reset отменит коммит, но не удалит его, а поместит все изменения этого коммита в рабочую директорию, так что с ними можно будет продолжить работать. HEAD~ означает "один коммит от последнего коммита". Если бы мы хотели удалить два последних коммита, то могли бы написать HEAD~2.
git reset HEAD~ откатывает последний коммит. СМОТРИ ОПИСАНИЕ ВЫШЕ.
