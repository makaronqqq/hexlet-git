git status показывает что сейчас идет в коммит, какие изменения сделаны
cd переходит в папку, cd .. переходит на папку вверх
git add . добавляет все изменения, можно добавить папку или файл или все файл в папке и тд
git branch показывает на какой мы ветке и какие ветки есть
git commit -m 'коментарий'  коммитит добавленный верез add изменения
git push отпрваляет на гитхаб версию с последнего коммита
rm удаляет файл
git rm удаляет файл и подготавливает его к коммиту (add не надо для него прописывать)
